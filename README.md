 Project Overview

This project demonstrates how to use **Polynomial Regression** and **Ridge Regression** to model and predict temperature patterns using a simple **Swat weather dataset**.

The goal is to understand:

* Non-linear relationships in data
* Overfitting in polynomial models
* How Ridge Regression helps reduce overfitting


 Dataset

The dataset contains simple weather information:

* **Day** → Input feature
* **Temperature_C** → Target variable
* **Humidity_Percent** → Additional feature
* **Rainfall_mm** → Additional feature



 Technologies Used

* Python 
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

---



Results

* Polynomial Regression fits the data with a curve
* Higher degree → better fit but risk of overfitting
* Ridge Regression produces a smoother curve and better generalization

---
Visualization

* Blue points → Actual data
* Red line → Polynomial Regression
* Green line → Ridge Regression



Model performance is evaluated using:
Mean Squared Error (MSE)

---

 Learnings

* Real-world data is often non-linear
* Overfitting is a major problem in ML
* Regularization techniques like Ridge are essential
